01 · 系统概览 System Overview

Pixel Heart OS 是一个以女主配置驱动全局生成、结合 Git 式记忆管理（Beads）与 AI 协作编辑的涌现型社会宇宙模拟系统。 用户通过自由描述、问卷引导或现实导入三种方式创建女主，AI 解析其灵魂结构后自动生成 NPC 网络与场景，并通过 LangGraph 多智能体引擎进行持续推演。

05 · Beads 记忆系统 Git-Style Memory Management

Beads 系统是 Pixel Heart OS 的核心创新，将 Git 版本控制的概念映射到叙事记忆管理， 实现了对话历史的分支探索、回溯与归因分析。每个 Bead 等同于一次 Git commit，携带情绪色彩与状态 diff。

Git 概念 Beads 对应 恋爱模拟含义 数据字段
Commit 记忆提交 每次对话/事件的永久记录 hash, parent, content, diff
Branch 平行可能 "如果我当时..." 的探索 branch_name, from_bead
Merge 融合 最佳路径的选择 strategy: best/combine
Diff 对比 关系变化的可视化 emotion_delta, affinity_delta
Tag 里程碑 第一次约会、第一次吵架 tags: ["milestone", "conflict"]
Revert 回溯 回到某个状态重新选择 checkout(bead_hash)
Blame 归因 关系破裂的责任分析 [(hash, author, change)]
